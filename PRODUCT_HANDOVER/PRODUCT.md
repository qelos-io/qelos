# Qelos

An AI-first application layer gateway where teams build, configure, and operate multi-tenant SaaS apps—data models, admin UI, AI agents, integrations, billing, and developer SDKs—from a single platform.

## Sections

- [API](api/PRODUCT.md) — REST endpoints for auth, data, AI, content, plugins, payments, and assets, unified through the gateway
- [Admin Frontend](frontend/PRODUCT.md) — Vue admin panel for building apps, managing users, and running plugin pages
- [SDK](sdk/PRODUCT.md) — TypeScript SDK for external app developers
- [Web SDK](web-sdk/PRODUCT.md) — Browser MFE bridge and embeddable AI chat widget
- [CLI](cli/PRODUCT.md) — Developer CLI for scaffolding, sync, and local dev
- [Integrators](integrators/PRODUCT.md) — Framework middleware that makes Qelos the API for any app
