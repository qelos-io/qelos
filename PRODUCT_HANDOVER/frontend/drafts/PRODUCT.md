# Drafts

Saved in-progress admin edits resumable across sessions.

## What Users Can Do

- **List drafts**: View all saved incomplete edits at `/drafts`
- **Resume draft**: Continue editing from where you left off
- **Delete draft**: Remove unwanted saved sessions

## Interface Elements

- **Drafts list** — Context display name, type, last saved, resume/remove actions
- **Auto-save** — Drafts created automatically during content box and form editing

## Draft Context

Each draft is scoped to:
- **User** — Only the author can access
- **Context type** — e.g. content box, configuration
- **Context ID** — Specific resource being edited
- **Route params** — Navigation state for resume
