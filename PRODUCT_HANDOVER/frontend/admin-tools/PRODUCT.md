# Admin Tools

Platform audit, impersonation, onboarding, and privileged addons.

## What Users Can Do

- **View events log**: Paginated audit trail at `/admin/log`
- **Filter events**: By period, user, workspace, kind, name, source, search
- **Impersonate users**: Act as another user with visible exit banner
- **Complete onboarding**: Quick Start wizard for new tenant setup
- **Use AI assistant**: Header drawer for natural-language app building

## Events Log

| Filter | Options |
|--------|---------|
| Period | Date range picker |
| User | User selector |
| Workspace | Workspace selector |
| Kind | Event kind dropdown |
| Name/Source | Event name and source filters |
| Search | Free-text search |

## Privileged Addons

Components visible to admin/editor roles:
- **AI assistant drawer** — Contextual SaaS builder chat
- **Assets panel** — File management in Manager mode
- **Impersonation selector** — Switch impersonated user/workspace

## Quick Start Wizard Steps

1. App name and logo
2. Theme selection
3. App type
4. OpenAI token (optional)
5. AI-generated blueprints
6. Initial setup completion
